Function prototypes are declared in this way :-

void myFunction(int);

Here's a program: 

            #include <iostream>
            using namespace std;

            bool isPrime(int n);    //function prototype
            int main()
            {
                int number = 5;
                bool ans = isPrime(5);
                if(ans == true)
                    cout << "Yes! " << number << " is a prime number";
                else 
                    cout << "No! " << number << " is not a prime number";
            }

In the above program

 bool isPrime(int n);

is a function prototype.
